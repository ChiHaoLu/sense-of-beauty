### 基本名詞

* 解析度：1080p ~ 2K
* 光圈（Aperture）：光圈越大散景越模糊，T 比 F 準確 e.g F5.6
* 偵數（fps）：影響流暢度，60 fps
* 快門（Shutter）：影響清晰度，1/60
* 感光（ISO）：絕對要 800 以內，最好 100~400
* 色溫：3000 或上下

> 最接近肉眼的動態模糊設定，「FPS * 2 = 需要的快門數」

### 記憶卡

1. 請直接使用 SD CARD，不要用 MicroSD 轉接
2. 品牌建議 Sandisk, Kingstone
3. 用電腦讀取照片的時候，請先把照片傳進電腦裡面，修改電腦裡面的圖檔不要修改記憶卡上的，即使只有將圖片轉向也請不要修改記憶卡上的檔案
4. 請使用相機格式化你的記憶卡，不要用電腦或其他的裝置格式化記憶卡
5. 將記憶卡拿到別的裝置用過再裝回相機，相機很有可能會顯示記憶卡錯誤，請使用相機格式化記憶卡
